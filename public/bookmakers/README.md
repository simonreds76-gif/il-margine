# Bookmaker Logos

Place bookmaker logo images in this folder.

## Required Logo Files

Create small thumbnail logos (recommended: 64x64px or 128x128px) for each bookmaker:

## Logo Specifications

- **Format**: PNG (with transparency preferred)
- **Size**: 64x64px to 128x128px (square)
- **Background**: Transparent or dark background
- **Style**: Simple, recognizable logo/icon

## Where to Find Logos

You can find bookmaker logos from:
- Official bookmaker websites (press/media sections)
- Logo databases (logos-world.net, brandfetch.com)
- Bookmaker affiliate portals (they often provide marketing assets)

## Notes

- Logos will be displayed at 24px (sm), 32px (md), or 40px (lg) depending on context
- If a logo is missing, a fallback with the first letter will be shown
- Logos are clickable if affiliate links are set in the database
