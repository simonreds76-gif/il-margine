# Bookmaker Logos

Place bookmaker logo images in this folder.

## Required Logo Files (Bookmakers page)

For the recommended bookmakers section, use these filenames (PNG or SVG):

- `midnite.png` (or .svg)
- `betvictor.png` (or .svg)
- `unibet.png` (or .svg)
- `coral.png` (or .svg)
- `ladbrokes.png` (or .svg)
- `betmgm.png` (or .svg)

Other bookmakers (if used elsewhere):

- `bet365.png`, `betfair.png`, `paddypower.png`, `williamhill.png`, `skybet.png`, `betway.png`, `888sport.png`, `betfred.png`, `pinnacle.png` (or `pinnacle.svg`), etc.

## Logo Specifications

- **Format**: PNG or SVG (PNG with transparency preferred)
- **Size**: **64×64 px or 128×128 px** (square). The bookmakers page displays logos at 32px; 64px gives a sharp 2× retina image, 128px gives 4× for extra sharpness.
- **Filename**: Use the exact name expected (e.g. `midnite.png`, `betvictor.png`, `unibet.png`, `coral.png`, `ladbrokes.png`, `betmgm.png`). The component also tries `.svg`, `.jpeg`, `.jpg` if the first fails.
- **Background**: Transparent or dark background
- **Style**: Simple, recognizable logo/icon

## Where to Find Logos

You can find bookmaker logos from:
- Official bookmaker websites (press/media sections)
- Logo databases (logos-world.net, brandfetch.com)
- Bookmaker affiliate portals (they often provide marketing assets)

## Notes

- **Bookmakers page** uses **32px (md)**. Recommended source size: **64×64 px** (or 128×128 px). Logos are shown at 24px (sm), 32px (md), or 40px (lg) depending on context.
- If a logo is missing, a fallback with the first letter will be shown
- Logos are clickable if affiliate links are set in the database
